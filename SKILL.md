# 小学作文评分技能

## 概述

这个技能用于从图片中识别手写作文内容，进行专业的评分分析，并提供详细的点评和修改建议。

**支持两种评分模式：**
1. **标准评分**：无作文要求时，使用通用评分标准
2. **按要求评分**：有作文要求图片时，根据要求针对性评分

## 工作流程

### 第零步：识别作文要求（如有）

如果用户提供了作文要求图片，先识别要求内容：

```python
from rapidocr_onnxruntime import RapidOCR

ocr = RapidOCR()
result, elapse = ocr(requirement_image_path)

if result:
    requirements = '\n'.join([line[1] for line in result])
```

**作文要求通常包含：**
- 作文题目/主题
- 字数要求
- 内容要求（如：写人、记事、状物、写景等）
- 结构要求（如：分段、首尾呼应等）
- 特殊要求（如：使用比喻、心理描写等）

**根据要求评分时需要：**
1. 检查是否符合题目要求
2. 检查是否达到字数要求
3. 检查是否包含指定的写作手法
4. 按要求权重调整评分标准

### 第一步：识别作文内容

使用 OCR 工具从图片中提取文字内容：

```python
# 使用 RapidOCR 进行中文手写体识别
# 需要先安装: pip install rapidocr_onnxruntime

from rapidocr_onnxruntime import RapidOCR

ocr = RapidOCR()
result, elapse = ocr(image_path)  # 注意：使用 __call__ 方法，不是 .ocr()

# 提取文字
if result:
    text_content = '\n'.join([line[1] for line in result])
else:
    text_content = ""  # OCR识别失败
```

**注意事项**：
- 手写体识别可能存在误差，需要人工校正
- 常见误识别：音近字、形近字、潦草字
- 需要根据上下文推断正确内容

### 第二步：整理作文内容

将 OCR 识别结果整理为：
1. **标题**
2. **正文段落**
3. **标注OCR误识别并修正**

### 第三步：评分分析

#### 评分维度与分值分配（满分30分）

| 维度 | 分值 | 评分要点 |
|------|------|----------|
| **内容质量** | 10分 | 切合题意、中心明确、内容充实、情感真实 |
| **语言表达** | 8分 | 语句通顺、用词准确、表达生动、无语病 |
| **结构层次** | 6分 | 条理清晰、段落分明、过渡自然、首尾呼应 |
| **书写规范** | 4分 | 字迹工整、标点正确、格式规范、错别字少 |
| **字数要求** | 2分 | 达到字数要求 |

#### 扣分细则

**1. 内容扣分（最高扣10分）**
- **偏离题意**：扣3-5分（严重偏题可降至及格线以下）
- **中心不明确**：扣2-3分
- **内容空洞**：扣1-2分
- **事例不具体**：扣1-2分
- **未体现人物特点**：扣2-3分

**2. 语言扣分（最高扣8分）**
- **病句**：每处扣0.5分，累计不超过3分
- **用词不当**：每处扣0.5分
- **语言干瘪**：扣1-2分

**3. 结构扣分（最高扣6分）**
- **条理混乱**：扣1-2分
- **段落不清**：扣1分
- **首尾不呼应**：扣1分

**4. 字数扣分（最高扣2分）**
- **字数不足**：
  - 少于400字：扣1分
  - 少于350字：扣2分
  - 少于300字：扣3分以上
- **字数超标过多**（超过600字）：一般不扣分，但若明显凑字数可扣1分

#### 按要求评分的重点检查

如果提供了作文要求，需要额外检查：

| 检查项 | 扣分标准 |
|--------|----------|
| 是否切题 | 偏题扣3-5分，跑题扣5-8分 |
| 人物选择是否恰当 | 选择职业英雄而非平凡人物，扣2-3分 |
| "身边"概念是否合理 | 偶遇的陌生人不算"身边"，扣1-2分 |
| 是否按要求使用写作手法 | 未按要求（如动作、语言描写），扣1-2分 |
| 人物刻画是否充分 | 缺乏外貌、动作、语言描写，扣2-3分 |

### 第四步：详细点评

#### 亮点分析（✨）

1. **立意**：分析主题是否深刻、积极
2. **比喻/修辞**：找出精彩的比喻、拟人等修辞手法
3. **心理描写**：分析心理变化的层次感
4. **环境描写**：分析环境烘托的作用
5. **结构**：分析首尾呼应、过渡自然

#### 不足之处（⚠️）

1. 指出具体的语句问题
2. 分析结构上的缺陷
3. 指出内容上的不足
4. 分析书写问题

### 第五步：修改建议

#### 格式

**原文**：
> 引用原文

**建议修改**：
> 修改后的内容

**修改理由**：
- 说明为什么这样改
- 改后的效果

### 第六步：总结

- 总分及等级
- 主要优点
- 主要改进方向
- 鼓励语

## 注意事项

1. **OCR识别局限**：手写体识别可能存在误差，需要：
   - 根据上下文推断正确内容
   - 标注明显的OCR误识别
   - 对关键语句进行人工确认
   - **推荐顺序**：RapidOCR > Tesseract > 视觉大模型

2. **OCR工具选择**：
   - **RapidOCR**：中文印刷体效果好，手写体一般
   - **Tesseract**：需要安装 chi_sim 语言包，命令：`tesseract image.jpg stdout -l chi_sim+eng`
   - **视觉大模型**：如 GPT-4V、Claude Vision，效果最好但需要 API

3. **评分核心原则**：
   - ⚠️ **必须严格依据题目要求评分**，不能主观臆断
   - ⚠️ **核心要求未满足要重扣分**（如题目要求"动作、语言描写"，但语言描写完全缺失，应扣5-6分）
   - ⚠️ **主次颠倒要扣分**：环境描写>人物描写，说明未抓住重点
   - ⚠️ **题目示例要参考**：题目说"如消防员、清洁工"，说明这些人物选择都符合题意，不能判定为偏题

4. **评分公正性**：
   - 参考小学生作文评分标准
   - 考虑年级水平（低年级适当放宽）
   - 注重鼓励和建设性意见

5. **点评建设性**：
   - 先肯定优点，再指出不足
   - 给出具体的修改建议
   - 语言温和鼓励

## 常见问题处理

### Q: OCR识别不出来怎么办？

尝试以下方法：
1. 使用多个OCR工具交叉验证
2. 对图片进行预处理（二值化、去噪）
3. 手动提示用户提供文字版

### Q: 字迹太潦草无法识别？

在评分中注明：
- 书写与卷面扣分
- 建议加强书写练习

### Q: 作文内容不完整？

根据识别到的部分内容进行评分，并在点评中说明情况。

### Q: 如何判断是否满足题目核心要求？

**关键步骤**：
1. 提取题目的核心要求（如"动作、语言描写"）
2. 逐一检查作文是否满足
3. **核心要求未满足要重扣分**
4. 示例：题目要求"动作、语言"，但语言描写完全缺失 → 内容质量扣5-6分

### Q: 如何判断"偏题"？

**重要原则**：
1. **题目示例的人物都符合题意**（如题目说"如消防员、清洁工"，写这些人物不算偏题）
2. **不能主观判定"职业英雄"不算"平凡英雄"**，要依据题目
3. 真正的偏题是：完全不符合题目要求的人物或主题

### Q: 环境描写过多是否扣分？

**判断标准**：
- 环境描写服务于人物刻画 → 不扣分
- 环境描写喧宾夺主，人物描写<20% → 结构层次扣1-2分
- 环境描写挤占了核心要求（如动作、语言描写）的内容 → 内容质量扣分

### Python 调用示例

```python
from rapidocr_onnxruntime import RapidOCR

def grade_essay(essay_image, requirement_image=None):
    """
    作文评分函数
    
    Args:
        essay_image: 作文图片路径
        requirement_image: 作文要求图片路径（可选）
    
    Returns:
        评分结果字典
    """
    ocr = RapidOCR()
    
    # 识别作文内容
    result, _ = ocr(essay_image)
    essay_text = '\n'.join([line[1] for line in result]) if result else ""
    
    # 识别作文要求（如有）
    requirements = None
    if requirement_image:
        req_result, _ = ocr(requirement_image)
        requirements = '\n'.join([line[1] for line in req_result]) if req_result else None
    
    # 评分分析
    if requirements:
        # 按要求评分模式
        score = score_with_requirements(essay_text, requirements)
    else:
        # 标准评分模式
        score = standard_score(essay_text)
    
    return score
```
