# 小学作文评分技能 v3.0

按标准评分细则对小学作文进行专业评分，支持多模态识图和并行评分优化。

## 核心更新（v3.0）

1. **多模态识图**：直接从图片识别并评分，无需单独 OCR
2. **并行优化**：专家模式 KIMI 和豆包 Vision 并行识图，速度提升 ~30%
3. **新增卷面分**：评分维度从 5 项增至 6 项（满分 32 分）

## 快速使用

### 普通模式（单模型识图）
```bash
python3 ~/.hermes/skills/productivity/essay-grading/scripts/essay_grader_v2.py <图片路径>
```

### 专家模式（并行多模型 + 仲裁）
```bash
python3 ~/.hermes/skills/productivity/essay-grading/scripts/essay_grader_v2.py <图片路径> --expert
```

## 评分策略

### 普通模式
```
KIMI-2.5 多模态识图 → 直接输出评分结果
```

### 专家模式（并行优化）
```
┌─────────────────────────────────────┐
│  并行执行                           │
│  ┌─────────────┐ ┌─────────────┐   │
│  │ KIMI-2.5    │ │ 豆包 Vision  │   │
│  │ 识图评分    │ │ 识图评分    │   │
│  └─────────────┘ └─────────────┘   │
└─────────────────────────────────────┘
              ↓
        GLM-5 仲裁
              ↓
        最终评分结果
```

## 评分标准（满分32分）

| 维度 | 分值 | 评分要点 |
|------|------|----------|
| 内容质量 | 10分 | 切合题意、中心明确、内容充实、情感真实 |
| 语言表达 | 8分 | 语句通顺、用词准确、表达生动、无语病 |
| 结构层次 | 6分 | 条理清晰、段落分明、过渡自然、首尾呼应 |
| 书写规范 | 4分 | 字迹工整、标点正确、格式规范、错别字少 |
| 字数要求 | 2分 | 达到字数要求（400字以上） |
| 卷面分 | 2分 | 书写整洁、卷面干净、无涂改痕迹 |

## 模型配置

| 模型 | 用途 | 特点 |
|------|------|------|
| `Kimi-K2.5` | 多模态识图评分 | 中文识别准确，评分专业 |
| `doubao-1-5-vision-pro-32k-250115` | 多模态识图评分 | 豆包视觉模型 |
| `GLM-5` | 专家模式仲裁 | 智谱旗舰，综合判断 |

详细配置见 `references/models.md`

## 输出格式

1. 字数统计
2. 符合题意检查
3. 分项评分
4. 详细点评（优点 + 不足）
5. 总分及总结
6. （专家模式）不同专家的打分及差异及评审意见

## 重要注意事项

1. **飞书输出**：⚠️ terminal 输出不会发送到飞书，必须直接在回复中输出内容
2. **评分核心**：必须严格依据题目要求，核心要求未满足要重扣分
3. **题目示例**：题目示例的人物都符合题意，不能主观判定偏题
4. **JSON解析**：GLM-5 可能返回被 \`\`\`json...\`\`\` 包裹的JSON，需要先去除代码块标记
5. **并行实现**：使用 `concurrent.futures.ThreadPoolExecutor` 并行调用多模态模型
