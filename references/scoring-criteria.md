# 评分标准与扣分细则

## 评分维度与分值分配（满分30分）

| 维度 | 分值 | 评分要点 |
|------|------|----------|
| **内容质量** | 10分 | 切合题意、中心明确、内容充实、情感真实 |
| **语言表达** | 8分 | 语句通顺、用词准确、表达生动、无语病 |
| **结构层次** | 6分 | 条理清晰、段落分明、过渡自然、首尾呼应 |
| **书写规范** | 4分 | 字迹工整、标点正确、格式规范、错别字少 |
| **字数要求** | 2分 | 达到字数要求 |

## 扣分细则

### 1. 内容扣分（最高扣10分）
- **偏离题意**：扣3-5分（严重偏题可降至及格线以下）
- **中心不明确**：扣2-3分
- **内容空洞**：扣1-2分
- **事例不具体**：扣1-2分
- **未体现人物特点**：扣2-3分

### 2. 语言扣分（最高扣8分）
- **病句**：每处扣0.5分，累计不超过3分
- **用词不当**：每处扣0.5分
- **语言干瘪**：扣1-2分

### 3. 结构扣分（最高扣6分）
- **条理混乱**：扣1-2分
- **段落不清**：扣1分
- **首尾不呼应**：扣1分

### 4. 字数扣分（最高扣2分）
- **字数不足**：
  - 少于400字：扣1分
  - 少于350字：扣2分
  - 少于300字：扣3分以上
- **字数超标过多**（超过600字）：一般不扣分，但若明显凑字数可扣1分

## 按要求评分的重点检查

| 检查项 | 扣分标准 |
|--------|----------|
| 题目是否正确 | 题目写错扣2-3分 |
| 是否切题 | 偏题扣3-5分，严重偏题扣5-7分 |
| 人物选择是否恰当 | 选择与题目示例性质不符的人物，扣2-3分 |
| 语言描写是否到位 | **题目要求"语言描写"却缺失，扣3-5分** |
| 动作描写是否具体 | 动作描写概括性（如只有"指挥着"），扣1-2分 |
| 人物刻画是否充分 | 缺乏外貌、动作、语言描写，扣2-3分 |
| 主次是否分明 | 环境描写>人物描写，喧宾夺主，扣1-2分 |

## 常见扣分问题汇总

### 严重问题（扣5分以上）
1. **题目错误**：题目写错，扣2-3分
2. **严重偏题**：内容与题目要求完全不符，扣5-7分
3. **核心要求未达标**：题目明确要求"语言描写"却完全缺失，扣3-5分

### 中等问题（扣2-4分）
1. **人物刻画不足**：缺乏外貌、动作、语言描写，扣2-3分
2. **人物选择争议**：选择的人物与题目示例性质不符，扣1-2分
3. **动作描写概括性**：只有"指挥着""打扫"等概括性动词，扣1-2分
4. **主次颠倒**：环境描写篇幅超过人物描写，扣1-2分

### 轻微问题（扣1-2分）
1. **字数偏多**：超过600字且有拖沓感，扣0.5-1分
2. **语句重复**：如"可...可..."重复使用，扣0.5分
3. **指代不明**：开头"他们""那个人"指代不清，扣0.5分
4. **结尾套话**：如"祖国正因为有他们..."等空泛总结，扣0.5分

## 等级划分

| 分数 | 等级 |
|------|------|
| 27-30分 | 一类文·优秀 |
| 24-26分 | 二类文·良好 |
| 18-23分 | 三类文·中等 |
| 12-17分 | 四类文·及格 |
| 0-11分 | 五类文·不及格 |
