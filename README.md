## 简介

基于多模态大模型的小学作文评分系统，支持手写作文图片识别、专业评分分析和详细点评。

## ✨ v2.0 新特性

### 双模式评分

| 模式 | 流程 | 耗时 |
|------|------|------|
| **普通模式** | KIMI-2.5 多模态识图评分 | ~60s |
| **专家模式** | KIMI ∥ 豆包Vision → GLM-5 仲裁 | ~90s |

### 并行优化

专家模式使用 `ThreadPoolExecutor` 并行调用两个视觉模型，节省约 30% 时间。

### 新增卷面分维度

评分维度从 5 项扩展为 6 项，满分从 30 分调整为 32 分。

## 功能特点

- 🖼️ **多模态识图**：直接识别手写作文图片，无需 OCR 预处理
- 🎯 **符合题意检查**：检查是否切题、写作手法使用情况
- 📊 **专业评分**：32 分制标准化评分（含卷面分）
- ✨ **亮点分析**：发现作文中的精彩之处
- ⚠️ **问题诊断**：指出需要改进的地方，给出扣分原因
- ⚖️ **专家仲裁**：多模型评分 + GLM-5 仲裁，结果更客观

## 快速开始

### 普通模式

```bash
python scripts/essay_grader_v2.py essay.jpg
```

### 专家模式

```bash
python scripts/essay_grader_v2.py essay.jpg --expert
```

## 评分标准（满分 32 分）

| 维度 | 分值 | 评分要点 |
|------|------|----------|
| 内容质量 | 10分 | 切合题意、中心明确、内容充实、情感真实 |
| 语言表达 | 8分 | 语句通顺、用词准确、表达生动、无语病 |
| 结构层次 | 6分 | 条理清晰、段落分明、过渡自然、首尾呼应 |
| 书写规范 | 4分 | 字迹工整、标点正确、格式规范、错别字少 |
| 字数要求 | 2分 | 达到字数要求（400字以上） |
| 卷面分 | 2分 | 书写整洁、卷面干净、无涂改痕迹 |

## 等级划分

| 分数 | 等级 |
|------|------|
| 29-32 分 | 一类文·优秀 |
| 25-28 分 | 二类文·良好 |
| 19-24 分 | 三类文·中等 |
| 13-18 分 | 四类文·及格 |
| 0-12 分 | 五类文·不及格 |
