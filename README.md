## 简介

小学作文评分分析技能，从图片中识别手写作文内容，进行专业的评分分析，并提供详细的点评和修改建议。

## 功能特点

- 📷 **OCR 识别**：支持手写作文图片识别
- 📊 **专业评分**：30 分制标准化评分
- ✨ **亮点分析**：发现作文中的精彩之处
- ⚠️ **问题诊断**：指出需要改进的地方
- 💡 **修改建议**：提供具体的修改方案
- 📈 **成长建议**：给出后续学习建议

## 快速开始

### 1. 识别作文内容

```python
from rapidocr_onnxruntime import RapidOCR

# 初始化 OCR
ocr = RapidOCR()

# 识别图片
result, elapse = ocr("essay.jpg")

# 提取文字
if result:
    text_content = '\n'.join([line[1] for line in result])
    print(text_content)
```

### 2. 评分分析

作文评分标准（满分 30 分）：

| 评分项目 | 满分 | 评分要点 |
|---------|------|---------|
| 内容与立意 | 10 分 | 主题明确、立意积极、内容充实 |
| 结构与条理 | 8 分 | 结构完整、段落分明、首尾呼应 |
| 语言表达 | 8 分 | 语句通顺、词汇丰富、描写生动 |
| 书写与卷面 | 4 分 | 字迹工整、卷面整洁、标点正确 |

### 3. 等级划分

| 分数 | 等级 |
|------|------|
| 27-30 分 | 优秀（A） |
| 24-26 分 | 良好（B） |
| 18-23 分 | 中等（C） |
| 12-17 分 | 及格（D） |
| 0-11 分 | 不及格（F） |

## 注意事项

1. **OCR 局限性**：手写体识别可能存在误差，需要人工校正
2. **评分公正性**：参考小学生作文评分标准，考虑年级水平
3. **建设性点评**：先肯定优点，再指出不足，语言温和鼓励
4. **隐私保护**：作文图片仅用于评分，不存储不外传

## 常见问题

### Q: OCR 识别不出来怎么办？
- 对图片进行预处理（二值化、去噪）
- 尝试多个 OCR 工具交叉验证
- 手动提示用户提供文字版

### Q: 字迹太潦草？
- 在"书写与卷面"项扣分
- 建议加强书写练习

### Q: 作文内容不完整？
- 根据识别到的部分评分
- 在点评中说明情况
